¿Cómo se hizo este portafolio?

Este portafolio fue creado utilizando tecnologías web estándar como HTML, CSS y JavaScript. El diseño es sencillo y enfocado en la experiencia del usuario.

Para la interacción, utilicé JavaScript para crear un reloj en tiempo real en el encabezado, así como para calcular los días desde una fecha específica.

El proyecto se organiza de manera modular, con secciones separadas para mostrar información sobre mí, mis proyectos y un formulario de contacto.
